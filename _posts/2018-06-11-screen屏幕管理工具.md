# Linux下Screen屏幕管理工具
**简单示例步骤：**
1. 创建一个新屏幕screenOne并进入：

```
screen -S screenOne 
```
2. 退出screen，结束会话：
```
exit
```
3. 保留会话，detached屏幕screen：
```
ctrl+a+d
```
4. 查看screen情况以及当前所在screen：
```
screen -ls
```
5. 恢复screen会话：
```
screen -r screenOne
```
